Playful interaction

Introductie

Deze week gaat het over playful interaction. Dit is naar mijn mening eigenlijk een grote samenvoeging is van alle voorgaande onderwerpen. Playful interaction is een interactieve manier van spelen en is heel breed wat betreft ‘regels’ en wat binnen playful interaction valt. Playful interaction kunnen namelijk samen zijn, maar ook alleen. Er kan een verhaallijn inzitten, maar het kan ook ongestructureerd zijn. Het kan verschillende manieren van input hebben en er moet altijd een vermakelijk element inzitten, maar de ervaring hoeft niet per se leuk te zijn.

Deze week ga ik een playful interaction conceptualiseren voor in de Haagse Hogeschool. Ook heb ik weer onderzoek gedaan naar een interessant project en hier vertel ik meer over.
